# QPSK Communication System Simulation

## Overview
This project implements a QPSK digital communication system over an AWGN channel and evaluates its performance using BER vs SNR analysis.

## Features
- QPSK modulation and demodulation
- AWGN noise simulation
- BER calculation
- BER vs SNR analysis

## Results
The BER decreases with increasing SNR, demonstrating improved reliability at higher signal quality.

## Technologies Used
- Python
- NumPy
- Matplotlib

## How to Run
1. Download the notebook
2. Run in Jupyter Notebook
